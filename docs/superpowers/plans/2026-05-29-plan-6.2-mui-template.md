# Plan 6.2: MUI (Material UI) Client Template

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `apps/templates/client-mui` as a third UI variant alongside `client-shadcn` (Plan 6) and `client-antd` (Plan 6.1). Same `libs/template-shared` foundation, same route tree, same `setNotifier` interface — only the visual layer differs. After this plan: `npm init @idevconn/icore my-app -- --ui=mui` no longer falls back to shadcn; the CLI ships all three UI libraries.

**Architecture:** Identical pattern to Plan 6.1. No new shared logic — `template-shared` already exposes everything the MUI template needs (auth store, i18n bootstrap, AbilityProvider, useNotify, useDraft, LandingPage). The MUI template wires its own visual layer:

- **Routes** — identical file tree to `client-shadcn` / `client-antd` (`/`, `/login`, `/_dashboard`, `/_dashboard/dashboard`, `/_dashboard/profile`).
- **Layouts** — MUI `Box` / `AppBar` / `Drawer` / `Toolbar` shell with Material Icons.
- **PageLayout** — MUI `Typography` header + `LinearProgress`/`Backdrop` loading overlay + `<Can>` gate.
- **LoginForm / ProfileForm** — MUI `TextField` + `Button` + `Box` with HTML5 validation.
- **AccessDeniedPage** — MUI `Container` + `Typography` + `Button` (status-403 visual).
- **Notify host** — MUI `Snackbar` via a custom provider, registered into `setNotifier`. Unlike antd's `notification.useNotification()` hook, MUI doesn't ship a singleton notifier — we build a thin queue + Snackbar host inside MainLayout.

**Tech Stack:** `@mui/material` ^6, `@mui/icons-material` ^6, `@emotion/react` ^11, `@emotion/styled` ^11, the same TanStack Router/Query, Zustand, i18next, CASL, useDraft pieces from `template-shared`.

**Source spec:** `docs/superpowers/specs/2026-05-28-icore-design.md` § Client UI library — bootstrap-time choice.

## Task 3: Bootstrap + landing route

### Steps

- [ ] **Step 1: Notify host (custom queue + Snackbar)**

MUI doesn't ship a singleton notifier; we wire one via a Zustand store + Snackbar host inside MainLayout.

Create `apps/templates/client-mui/src/lib/notify.ts`:

```ts
import { create } from 'zustand';
import { setNotifier, type NotifyOptions } from '@icore/template-shared';

export type NotifyVariant = 'success' | 'error' | 'info' | 'warning';

export interface NotifyToast {
  id: number;
  variant: NotifyVariant;
  title: string;
  description?: string;
  duration: number;
}

interface NotifyState {
  queue: NotifyToast[];
  push: (variant: NotifyVariant, title: string, opts?: NotifyOptions) => void;
  dismiss: (id: number) => void;
}

let nextId = 1;

export const useNotifyStore = create<NotifyState>((set) => ({
  queue: [],
  push: (variant, title, opts) =>
    set((state) => ({
      queue: [
        ...state.queue,
        {
          id: nextId++,
          variant,
          title,
          description: opts?.description,
          duration: opts?.duration ?? 4000,
        },
      ],
    })),
  dismiss: (id) => set((state) => ({ queue: state.queue.filter((t) => t.id !== id) })),
}));

export function wireMuiNotifier(): void {
  setNotifier({
    success: (title, opts) => useNotifyStore.getState().push('success', title, opts),
    error: (title, opts) => useNotifyStore.getState().push('error', title, opts),
    info: (title, opts) => useNotifyStore.getState().push('info', title, opts),
    warning: (title, opts) => useNotifyStore.getState().push('warning', title, opts),
  });
}
```

- [ ] **Step 2: Snackbar host component**

Create `apps/templates/client-mui/src/components/NotificationHost.tsx`:

```tsx
import { Alert, Snackbar } from '@mui/material';
import { useNotifyStore } from '../lib/notify';

export function NotificationHost() {
  const queue = useNotifyStore((s) => s.queue);
  const dismiss = useNotifyStore((s) => s.dismiss);

  return (
    <>
      {queue.map((toast, idx) => (
        <Snackbar
          key={toast.id}
          open
          autoHideDuration={toast.duration}
          onClose={() => dismiss(toast.id)}
          anchorOrigin={{ vertical: 'top', horizontal: 'right' }}
          // Stack toasts visually
          sx={{ mt: idx * 7 }}
        >
          <Alert
            onClose={() => dismiss(toast.id)}
            severity={toast.variant}
            variant="filled"
            sx={{ width: '100%' }}
          >
            <strong>{toast.title}</strong>
            {toast.description ? <div>{toast.description}</div> : null}
          </Alert>
        </Snackbar>
      ))}
    </>
  );
}
```

## Task 6: `PageLayout` + `AccessDeniedPage`

`apps/templates/client-mui/src/components/PageLayout.tsx`:

```tsx
import type { ReactNode } from 'react';
import { Box, LinearProgress, Stack, Typography } from '@mui/material';
import { Can, useDraft, useLoading } from '@icore/template-shared';
import { AccessDeniedPage } from './AccessDeniedPage';
import type { AbilityAction, AbilitySubject } from '@icore/shared';

export interface PageLayoutProps {
  title: ReactNode;
  description?: ReactNode;
  action?: AbilityAction;
  subject?: AbilitySubject;
  extra?: ReactNode;
  children?: ReactNode;
}

export function PageLayout({
  title,
  description,
  action = 'read',
  subject = 'all',
  extra,
  children,
}: PageLayoutProps) {
  useDraft(false);
  const loading = useLoading();

  return (
    <Can I={action} a={subject} passThrough>
      {({ isAllowed }) =>
        isAllowed ? (
          <Box sx={{ p: 3 }}>
            <Stack
              direction="row"
              justifyContent="space-between"
              alignItems="flex-start"
              spacing={2}
              mb={3}
            >
              <Box>
                <Typography variant="h4" component="h1">
                  {title}
                </Typography>
                {description ? (
                  <Typography variant="body2" color="text.secondary" mt={0.5}>
                    {description}
                  </Typography>
                ) : null}
              </Box>
              {extra ? (
                <Stack direction="row" spacing={1}>
                  {extra}
                </Stack>
              ) : null}
            </Stack>
            {loading ? <LinearProgress sx={{ mb: 2 }} /> : null}
            <Box>{children}</Box>
          </Box>
        ) : (
          <AccessDeniedPage />
        )
      }
    </Can>
  );
}
```
